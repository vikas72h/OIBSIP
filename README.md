# OIBSIP_DataScience_Task1

Iris Flower Classification using Machine Learning: Developed a Machine Learning model to classify Iris flower species into Setosa, Versicolor, and Virginica based on sepal and petal measurements. The project involved data exploration, visualization, model training, and performance evaluation. 🌸📊

Objective: Build a Machine Learning model to accurately classify iris flower species using flower measurements.

Tools & Technologies:

1. Python
2. Pandas
3. NumPy
4. Matplotlib
5. Seaborn
6. Scikit-learn

Steps Performed:

#Loaded and explored the Iris dataset.
#Performed data preprocessing and cleaning.
#Visualized feature distributions and species relationships.
#Selected relevant features for classification.
#Split the dataset into training and testing sets.
#Trained a Random Forest Classifier model.
#Predicted flower species using the trained model.
#Evaluated model performance using Accuracy Score and Classification Report.

Machine Learning Model:

#Random Forest Classifier

Outcome: Successfully developed a Machine Learning model that accurately classifies Iris flower species and achieved high prediction accuracy on the test dataset.
